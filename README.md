# Enosx Technologies - E-commerce Aggregator

## Overview

Enosx Technologies is a trusted electronics aggregator, bringing you the best deals from Kenya's top e-commerce platforms like Jumia, Kilimall, and Jiji all in one place.

## Features

- **Unified Search**: Search for products across multiple platforms simultaneously.
- **Price Comparison**: Compare prices and features to find the best deals.
- **Seamless Ordering**: Place orders directly through our integrated platform.
- **Order Tracking**: Keep track of all your orders in one dashboard.

## Deployment

The project is live at:
**[https://enosxtech-hub.vercel.app](https://enosxtech-hub.vercel.app)**

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm or npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Enosx-X/enosxtech-hub.git
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Run the development server:
   ```bash
   pnpm dev
   ```

## Technology Stack

- **Frontend**: Next.js, React, Tailwind CSS
- **UI Components**: Radix UI, Lucide React
- **Backend**: Next.js API Routes
- **Database**: Supabase
